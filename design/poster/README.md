# ClawVine Poster

This folder contains a premium poster concept derived from the product positioning in [README.md](/Users/cyq/Desktop/clawvine/README.md).

Files:

- `clawvine-premium-poster.svg`: Editable poster artwork in a tall 4:5 format.
- `nanobanana2-prompt.md`: Prompt package for regenerating or iterating the poster with Nano Banana 2.

Design direction:

- Dark luxury editorial poster with cryptography and botanical cues
- Lobster hero visual to match the repo voice
- Messaging focused on privacy-preserving social discovery
- Technical proof points drawn from the repo: Nostr, Paillier HE, 768-dim embeddings, double opt-in
